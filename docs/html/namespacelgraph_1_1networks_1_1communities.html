Computes the connected components of this graph.

Parameters
[in]gThe input graph.
[out]compscomps[i] = j if, and only if, the i-th vertex belongs to the j-th connected component, where i is a node index.
The labels start at 0. Any label's value greater or equal than the number of vertices of g is not valid.
[out]binsIf this parameter is not null then bins[i] contains how many vertices are in the i-th connected component.
Returns
Returns the amount of connected components in g.
