Detailed Description

Read/write graphs in digraph6 format.

This format is detailed (as of 20th October 2018) in the webpage:

http://users.cecs.anu.edu.au/~bdm/data/formats.html

by Brendan McKay (http://users.cecs.anu.edu.au/~bdm/)

Reads an unweighted, directed graph in digraph6 format.

The file can only contain one single graph. If there is more than one, only the first will be read.

Parameters

[out]gThe graph built with the contents in the file.
