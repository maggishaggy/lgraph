Detailed Description

Collection of utilities.

Includes the data structures for:

  • graphs (see xxgraph, uxgraph, uugraph, udgraph, wxgraph, wugraph, wdgraph)
  • paths in graphs (see node_path, boolean_path)
  • bitsets (see static_bitset)
  • wrappers for random number generators (see random_generator)

plus definitions for the types to represent a node, the neighbourhood of a node in a graph, an edge, ...
