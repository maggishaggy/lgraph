Detailed Description

Abstract class for the graph data structure.

This interface requires the implementation of several methods to complete the implementation of a data structure for graphs that uses adjacency lists. It has only two attributes: adjacency_list, that stores the adjacency list of each node, and num_edges that contains the number of edges in the graph. Depending on the type of graph implemented these two attributes should be modified accordingly.
