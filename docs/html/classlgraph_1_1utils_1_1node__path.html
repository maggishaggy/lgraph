Detailed Description

template<class T = _new_>
class lgraph::utils::node_path< T >

A path through a grpah seen as a list of nodes.

It simply stores a list of nodes in which every pair of consecutive nodes form an edge in a graph. If all consecutive pairs of nodes make a path in a graph G then this object stores a path within G.

It also stores the length of the path, that is, the sum of all the edges in this path.

Parameters
TThe type used for the length. It must support comparisons and the C++'s output operator <<.
