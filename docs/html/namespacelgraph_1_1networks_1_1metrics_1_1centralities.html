Betweenness centrality of a node.

This centrality measure is implemented following the recommendations given in [Newman, 2010]:

  • Any path from vertex 's' to vertex 't' is considered to include both endpoints.
  • The equation to calculate the centrality also considers paths from any vertex to itself.

Furthermore, only the pairs of vertices (s,t) with s <= t are considered. Finally, this metric is also normalised.

[Newman, 2010] Newman, M. E. J. (2010). Networks. An introduction. Oxford University Press, Oxford.

This function computes all shortest paths between each pair of nodes.
