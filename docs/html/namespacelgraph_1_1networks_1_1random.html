Barabasi-Albert model: No vertex growth variant.

Similar to the BA_preferential_attachment model. This time, it differs from it in that the initial graph has the same number of vertices as the final graph: n0.

The T steps are used to make m0 connections at each step. More precisely:

  • Create an empty graph with n0 vertices
  • For step ti = 1 to T
    • Choose a vertex with probability proportional to its degree. Let u be that vertex.
    • Make m0 connections with some of the other vertices in the graph. Each edge (u, v) is added with probability proportional to the degree of v.

Barabasi-Albert model: Preferential attachment variant.

Builds a graph with n0 + T vertices. The first step is to build an empty graph with n0 vertices.

Then add T new vertices, each of which is connected to m0 already existing vertices of the graph each with a probability proportional to their degree.

Watts & Strogatz's model.

Create a cycle graph of N nodes, then connect each node with its k nearest nodes in the ring (or k - 1 if k is odd).

The rewiring is done by iterating over all edges created in the previous two steps (u,v) and replacing it with an edge (u,w) with probability p.

w is chosen uniformly at random from the set of non-neighbours of u at the moment of rewiring.

Parameters
crgContinuous random number generator. Has to be seeded by the caller.
drgDiscrete random number generator. Has to be seeded by the caller.
NNumber of nodes of the graph.
kNumber of the k nearest neighbours in a ring topology. Must satisfy: 2 <= k <= N - 1.
pProbability of rewiring an edge.
