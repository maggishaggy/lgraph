Detailed Description

Definition of a generic BFS traversal of an weighted graph.

Contains a generic Dijkstra algorithm that uses three different functions that guide it.

Generic Dijkstra algorithm.

The procedure applied for this algorithm here for traversing through a graph from a source node to a target node is as follows:

Dijkstra(graph, source, target):
  1.    vis = {false}   // set of |V(graph)| bits set to false
  2.    Q = empty       // empty priority queue of (distance,node)
  3.    while Q is not empty do
  4.        (wu,u) = Q.front
  5.        remove Q's front
  6.        if u is not visited then
  7.            proc_curr(G,u,vis)
  8         if terminate(G,u,vis) then terminate
  9.            else
 10.                Nu = neighbourhood of u
 11.                for each v in Nu do
 12.                    wv := weight edge (u,v)
 13.                    add_next := proc_neigh(G, u,v, wv, vis)
 14.                    if add_next then
 15.                        push w into Q
 16.                    endif
 17.                endfor
 18.            endif
 19.        endif
 20.    endwhile
Parameters
GThe graph being traversed
sourceThe node where the algorithm starts at
termThe terminating function. It is used as a termination condition in line 8.
proc_currThe function to process the currently visited node. It is called in line 7 used to perform some operation on the current node of the traversal.
proc_neighIt is called between the 'for' loop in line 13 and the used to perform some operation on each of the neighbours.
