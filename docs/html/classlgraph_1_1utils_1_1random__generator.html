Detailed Description

template<class G, typename T>
class lgraph::utils::random_generator< G, T >

Abstract pseudo-Random Number Generator (ARNG).

Interface for random number generators using the C++11 header <random>.

Seed the random engine calling method seed_random_engine().

Initialise whatever distribution is needed with the appropriate method:

  • init_uniform(T a, T b) to generate numbers following the distribution U[a,b], where U denotes the uniform distribution.
  • init_binomial(T n, double p) to generate numbers following the distribution B(n,p), where B denotes the binomial distribution.
Parameters
GThe random engine used to generate the numbers
TThe type of the numbers generated (its type depends on the subclass that will implement the interface)
